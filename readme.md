# wongoose网络库
## 目的
**使用IOCP重构mongoose网络库，旨在为C2提供方便和足够性能的多协议网络库)**
## 特点
- 更好的性能，使用iocp支持更高的并发量
- 多协议，根据mongoose的代码开发多协议通信，并提供更多更为方便的API使用
- 更好的多线程支持，收发等缓冲区不再如履薄冰，多线程处理数据不会引起崩溃，乱序及阻塞的情况
- 方便-你只需要包含wongoose.h并添加lib链接即可
